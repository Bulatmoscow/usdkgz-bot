# USDKG Telegram Bot

AI агент для создания презентаций USDKG через Telegram + Gamma App.

## Переменные окружения (добавить в Railway)

```
TELEGRAM_BOT_TOKEN=твой_токен_от_BotFather
ANTHROPIC_API_KEY=твой_ключ_от_Anthropic
```

## Команды бота

- /start — начало работы
- /help — справка
- /full — полная презентация (15 слайдов)
- /investor — питч для инвестора (8 слайдов)
- /exchanger — питч для обменников (5 слайдов)
- Любой текст — агент отвечает и создаёт контент

## Как использовать с Gamma

1. Получи текст от бота
2. Открой gamma.app → New → Paste text
3. Gamma автоматически создаёт слайды
4. Выбери тёмную тему с золотым акцентом
